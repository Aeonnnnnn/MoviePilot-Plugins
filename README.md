# MoviePilot 弹幕刮削插件

基于 [HankunYu/MoviePilot-Plugins](https://github.com/HankunYu/MoviePilot-Plugins) 弹幕刮削 v1.9.0 定制增强版。

## 功能

- 弹幕刮削（弹弹play API → ASS文件）
- TMDB媒体识别匹配
- 手动番剧匹配（目录/文件级别，支持集数偏移）
- 批量刮削、多线程并发
- 七阶段弹幕内容过滤系统（模式屏蔽/关键词/信用评分/相似弹幕/同屏密度删减）
- 批量分季匹配下载
- Vue 前端配置界面
- 侧栏导航（刮削管理、过滤管理）

## 安装

在 MoviePilot 插件市场的第三方仓库设置中添加本仓库地址即可。

## 仓库地址

```
https://github.com/你的用户名/你的仓库名
```
